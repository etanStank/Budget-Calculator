# Budget Calculator
> This is my new budget calculator project. This Budget Calculator allows for someone to get plan their budget through a specific time period. This is the second iteration of this project, the previous iteration can be found [here](https://github.com/etanStank/Budget-Planner).
> 
> This Budget Calculator will have a different structure from the previous iteration, as well as new code.

### Features
- Add one-off items.
- Add a specific income / expense (Weekly, Monthly).
- Add items with a income / expense over an inputted time period.
